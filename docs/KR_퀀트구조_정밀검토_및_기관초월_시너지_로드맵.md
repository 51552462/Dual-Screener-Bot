## 0. 한눈에 보는 결론 (Executive Summary)

현재 KR 시스템은 **"적응형 국면 엔진 + 행동주의(behavioral) 청산 + 전략 생존 토너먼트 + 한국형 대체데이터 엣지"** 를 갖춘, 개인 단위로는 매우 드문 고도화 시스템이다. 특히 다음 3가지는 이미 **상당수 중소형 헤지펀드를 능가**한다.

1. **진짜 학습하는 국면 가중치** — 5팩터 가중치가 5일 선행 실현손익으로 강화학습(시간분리, 룩어헤드 없음).
2. **행동주의 청산 엔진** — 유동 분할청산(fluid scale-out) + 볼록 트레일링 래칫(convex ratchet) + 피라미딩 + 거래량 확인 트레일링. 대부분의 룰베이스 시스템보다 정교.
3. **한국 특화 대체데이터** — 외국인·기관 순매수(pykrx) 칼만 평단 추정, 상한가 DNA 포렌식, 슈퍼노바 DNA 템플릿.

그러나 **기관을 "초월"하려면** 다음 4개 구조적 공백을 메워야 한다. 이것이 본 문서의 핵심이다.

| 공백 | 한 줄 요약 | 영향도 |
|---|---|---|
| **A. 포트폴리오 리스크 모델 부재** | 공분산·상관·VaR·변동성 타게팅이 전혀 없음 → "개수 기반" 분산만 존재 | ★★★★★ |
| **B. KR 팩터 빈곤 / 프록시 불일치** | KR은 VKOSPI 대신 미국 VIX를 빌려쓰고, breadth(시장폭) 팩터가 아예 없음 | ★★★★☆ |
| **C. 이미 받아오지만 버리는 데이터** | 외국인·기관 순매수를 매일 받아오지만 **시계열로 저장 안 함**(스냅샷만) | ★★★★☆ |
| **D. 검증 엄밀성 / 과최적화 위험** | walk-forward·OOS 분리·다중검정 보정 없음 → false discovery 위험 | ★★★★☆ |

> **핵심 통찰**: 기관은 자본·인프라·집행속도로 이긴다. 개인이 기관을 이기는 길은 그들과 같은 게임(초단타·대형주 효율시장)이 아니라, **기관이 들어올 수 없는 비대칭 영역**(KR 소형주, 상한가 DNA, 리테일/수급 타이밍)을 극대화하면서, 동시에 **기관 수준의 리스크 관리(공분산·변동성 타게팅)** 를 얹는 것이다. 본 시스템은 전자(엣지)는 강하나 후자(리스크 통제)가 약하다. → **"엣지는 유지, 리스크는 기관화"** 가 전략의 뼈대.

### 1.2 국면 / 팩터 엔진 + 적응 학습

**`predictive_regime_ensemble.py`** — 5팩터를 융합해 시장당 BULL/SIDEWAYS/BEAR 판정 후, 실현손익으로 가중치를 자가진화한다.

**5팩터 (`FACTORS`, line 49)** 와 점수식 (`compute_factor_states:322-339`, 모두 tanh로 [-1,+1] 클램프):

| 팩터 | 점수식 | KR 적용 |
|---|---|---|
| short_trend | `tanh(((close/ma20)-1)/0.03)` | ✅ |
| long_trend | `tanh(((close/ma200)-1)/0.08)` (※ ma200은 실제로 **EMA(200)**) | ✅ |
| vix | `-tanh((vix-18)/8)` (+크립토 카나리아 페널티 차감) | ⚠️ **미국 VIX 차용** |
| breadth | `tanh((breadth_ratio-1)/0.03)` | ❌ **KR은 None → 미투표** |
| pri | `clamp(pri_z, -1, +1)` | ✅ (주간 PRI) |

- **가중합 → 국면**: 결측 팩터 제외 후 재정규화한 가중평균 score. `score≥0.18 → BULL`, `≤-0.18 → BEAR`, 그 외 SIDEWAYS (`_regime_from_score:353`).
- **가중치 제약**: 거시 닻 하한(long_trend+vix ≥ 0.15), PRI 단일 상한 0.85 (`project_weights:190-231`) → 파멸적 망각 차단.
- **위기 오버라이드**: VIX≥30 또는 (VIX>60일p90 & VIX≥20) → 하드 위기. 미국 위기 시 KR을 HIGH_VOL로 강제 동기화(비대칭, `_stress_merge`).
- **히스테리시스**: 국면 전환은 n일 연속 관측 후 확정(위기는 우회). n일도 RL 튜닝(1~5일 클램프).
- **켈리 클러치**: 시장간 최소 top-국면 확률 < 0.60 → 전역 켈리 ×0.10~0.30 축소.

**진짜 학습하는 부분 (genuinely adaptive):**
1. **가중치 진화 `evolve_weights:639-721`** — 매일 팩터 상태 기록 → 5일 후(`FWD_EVAL_DAYS=5`) 실현손익으로 보상 `reward_f = state_f·tanh(fwd_pnl%/4)` → skill EMA(0.8/0.2) → `softmax(skill/2.2)` → project. **시간분리·룩어헤드 없음. 방법론적으로 건전.**
2. **히스테리시스 진화 `evolve_hysteresis`** — 휩쏘 손실 시 ↑, 이익 시 ↓.
3. **둠스데이 γ `doomsday_dampener.py`** — 주간 경사하강으로 방어강도 자가조정.

**휴리스틱(하드코딩) 부분:**
- 팩터 점수식·임계치(0.18, VIX 18/8/30) = 고정.
- **메타거버너 `meta_governor.py`** = ML이 아니라 7단계 휴리스틱 보정 파이프라인(롤링 분위수 + 하드코딩 임계). 특히 국면→자본 레버는 **고정 테이블** `ACTION_BY_REGIME`(line 43-71):

```
HIGH_VOL: kelly_cap 0.012   BEAR: 0.01   BULL: 0.028   SIDEWAYS: 0.018
```

→ **KR 사이징에 가장 큰 영향을 주는 단일 숫자(국면별 kelly_cap)가 학습되지 않은 매직 상수.**

- 전략 라이프사이클(승격/강등) = 결정론적 임계 상태기계. 트레저리 건강도 = wr/pf/mdd/streak 컷오프.
- **실제 자본배분기**는 `evolution/deathmatch_allocation.py`(배틀로얄 랭킹 → 그룹 켈리 배수, 챔피언 1.35×/하위 0.0). (혼동주의: 루트 `capital_deathmatch.py`는 **리포팅 비교기**일 뿐 배분 안 함.)

**유일한 OOS 검증**: `mutant_oos_validator.py` — 합성 챔피언을 실제 OHLCV로 익일수익률 검증(excess_alpha>0.00005 & wr>0.50 & n≥30). 단, **같은 DB 최근 6개월**(in-sample 육종과 시간 간격 없음), 다중검정 보정 없음 → false discovery 위험.

### 1.3 자본배분 / 사이징 / 리스크 / 청산

**사이징 (실제로는 ATR 리스크패리티, "진짜 켈리 아님")** — `forward/shared.py` `try_add_virtual_position`:

```
shares = floor( group_seed × kelly_risk_pct / (ATR_SL_mult × ATR) )
notional = shares × entry_price
```

- ATR_SL=2.0(기본), ATR=14기간 EWM. → 포지션당 `kelly_risk_pct`만큼을 2×ATR 손절까지 리스크. **변동성은 주수에만 반영**(포트폴리오 변동성 타게팅 없음).
- `kelly_risk_pct`는 base(기본 1%)에 **긴 배수 체인**: 전략가중(WEIGHT_S1/S4) × 톰슨샘플링(Beta) × 로테이션(2×) × 스필오버(1.5×) × 제네시스 × 합성DNA(1.5×) × 메타거버너(global×ns×group) × 템플릿 밴딧.
- 캡: 포지션당 `min(MAX_POSITION_PCT 25%, 메타캡) × seed`, 현금 브레이크(가용현금≤0 기각), **KR 소형주 브레이크(seed>5천만 & 시총<1000억 기각)**.
- **독립 복리 시드**: 각 시그널 로직이 자기 가상북 `seed = ACCOUNT_SIZE(₩2천만) + 자기 실현손익`. → 단일 통합 NAV가 아니라 **로직별 펀드매니저 모델**. (단, NAV 회계는 `live_nav_manager`가 `BASE_CAPITAL[KR]=₩3억`을 씀 → **자본 기준 이원화**.)

**청산 엔진 (`forward/ledger.py` `track_daily_positions`)** — 장중 고저 기반 우선순위 캐스케이드:
- **P1 절대 MAE/MFE**: 손절 -3.5%(MAE), 익절 +10%(MFE, v_energy≥20이면 ×1.10 오버드라이브). 익절 시 하드캡이 아니라 **유동 분할청산**(방어국면 ~78% / 불 ~18% / 횡보 ~45%).
- **P1b 프리러너 볼록 래칫**: 트레일 = MaxHigh×(1-κ), κ=0.05~0.12(런업 클수록 타이트). **거래량 확인**(ICR≥0.25 or RVOL≥1.5) 없으면 "가짜 하락"으로 청산 보류. κ는 주간 RL 튜닝.
- **P1c 피라미딩**: 프리러너+불+엣지>1.5 → 유휴 NAV에서 최대 3회 추가(각 ≤NAV 10%).
- **P2 추세/시간 손절**: 시간손절 10일(엣지>1.5면 +2일), ATR손절 2×, 테크손절(ZLEMA/EMA크로스). 모드 TECH/STAT/HYBRID(기본).
- **P3 좀비 강제청산**: bars_held≥2×시간손절.
- **시장폭 붕괴 긴급 타이트닝**: breadth<0.97 → 시간손절 ×0.5, 손절 ×0.5.

### 1.4 데이터 레이어 + 분석/리포팅

**가격데이터**: **일봉만**(FDR→yfinance→sqlite, KR 공통주 ~2,483종 + KOSPI/KOSDAQ 지수 프록시, ~1000영업일). 분/틱/호가/수정주가 없음.

**이미 보유한 대체데이터 엣지:**
- **외국인·기관 순매수** `smart_money_tracker.py` — pykrx 투자자별 순매수 리더보드 + 네이버 폴백, Top-55 가격괴리 필터, **칼만필터 평단 추정**. → `SMART_MONEY_RADAR` JSON. ⚠️ **단, 시계열 미저장(스냅샷만)**.
- **상한가 DNA 포렌식** `limit_up_forensics.py` — 당일 상한가의 T-10~T-2 사전패턴 10플래그(거래량압축, MA수렴, 사전스퀴즈 등) → 코호트 합의룰(≥70% 투표).
- **슈퍼노바 플로우** `Supernova_Flow_Tracking_Master.csv` (OHLCV 파생 DNA).
- **뉴스 감성** `news_data.sqlite` — Gemini 2.5-flash가 헤드라인만 0~100 채점. **시장 전체 1점/일(종목별 없음)**.

**부재 데이터 (검증됨):** 펀더멘털(PER/PBR/ROE/실적), **DART 공시**, 공매도/대차잔고, KR 주식 옵션/선물 포지셔닝, VKOSPI, 분/틱/호가, 종목별 뉴스 감성, 수정주가.

**리포팅**: 일일 9패널 종합(승률+PF, 켈리vs고정 데스매치, DNA 부검, 섹터 로테이션, 메타거버너 라이프사이클, 카나리아 헷지), 딥다이브(90/180일), 주간/월간 롤업. **계산함**: 승률, PF, 실현손익, MDD/드로다운 방어. **미계산**: **Sharpe/Sortino/Calmar 없음**, **팩터노출 어트리뷰션(Fama-French류) 없음**.

## 2. 기관·헤지펀드 대비 냉정한 갭 분석

> 기관 퀀트의 4대 무기: ① 포트폴리오 리스크 모델(공분산·팩터리스크) ② 멀티팩터 폭(가치·퀄리티·모멘텀+대체) ③ 집행/마이크로구조 ④ 검증 엄밀성. 아래는 본 시스템의 위치.

| 영역 | 기관 표준 | 본 시스템 현재 | 갭 | 비고 |
|---|---|---|---|---|
| 국면 적응 | 레짐 스위칭 모델 | **RL 가중치 진화(시간분리)** | ✅ **동등~우위** | 개인 강점 |
| 청산/출구 | 룰 또는 옵션헤지 | **유동분할+볼록래칫+피라미딩** | ✅ **우위** | 개인 강점 |
| 전략 생존 | 자본배분위원회 | 배틀로얄 토너먼트 | ✅ 동등 | |
| 대체데이터(KR) | 고가 벤더 | **수급/상한가 DNA 자체구축** | ✅ 비대칭 엣지 | 개인 강점 |
| **포트폴리오 리스크** | **공분산·VaR·변동성타게팅** | **개수 기반 분산만** | ❌ **치명적 공백** | §3-A |
| **멀티팩터 폭** | 가치+퀄리티+모멘텀+수급 | **거의 순수 기술적** | ❌ 큰 공백 | §3-C/D |
| **시장폭/변동성(KR)** | VKOSPI·A/D·신용 | **미국 VIX 차용, breadth 없음** | ❌ 프록시 불일치 | §3-B |
| **검증 엄밀성** | walk-forward·purged CV·DSR | 단일 익일 OOS, 보정 없음 | ❌ 과최적화 위험 | §3-E |
| 성과지표 | Sharpe/Sortino/팩터귀속 | 승률/PF/MDD만 | ⚠️ 부분 공백 | §3-F |
| 집행 | VWAP/TWAP/충격모델 | 일봉, 0.5% 슬리피지 가정 | ⚠️ (가상매매라 영향 작음) | §3-G |

## 3. 시너지 / 추가 제안 — 우선순위 로드맵

각 항목: **문제 → 제안 → 구현 위치 → 기대효과 → 난이도/리스크**. 모든 제안은 **기존 파일기반·무DB락·방어적(graceful degradation)** 원칙을 따르며, 기존 알파를 훼손하지 않는 "오버레이/추가" 방식이다.

---

### 🟥 P0 (즉효·고임팩트·저위험) — 먼저 할 것

#### P0-1. 외국인·기관 순매수 **시계열 영속화** → "수급 팩터" 신설
- **문제**: `smart_money_tracker`가 매일 pykrx로 진짜 수급을 받아오지만 **스냅샷 JSON만** 남기고 시계열을 버린다. 가장 값싼(이미 받는) 최고급 엣지를 낭비 중.
- **제안**:
  1. `market_data.sqlite`에 `kr_investor_flow(date, code, foreign_net, inst_net, retail_net, foreign_cum20, inst_cum20)` 테이블 신설, 매일 16:10 적재.
  2. 파생 팩터: **수급 모멘텀**(5/20일 누적 순매수), **수급 다이버전스**(가격은 횡보인데 외국인·기관 동반 순매수), **연속 순매수일수**.
  3. 진입 가산점 + 5팩터 엔진의 KR 신규 팩터 후보(`smart_flow`)로 승격.
- **구현 위치**: `smart_money_tracker.py`(적재 추가), 신규 `kr_flow_factor.py`, `predictive_regime_ensemble.py`(팩터 후보), `forward/shared.py`(진입 가산).
- **기대효과**: KR에서 외국인·기관 수급은 **단일 최강 선행지표**. 종목선정 정밀도↑, 국면판정에 KR 고유신호 주입(미국 VIX 의존도↓).
- **난이도**: 중 / **리스크**: 낮음(읽기 적재, 기존 로직 무영향). 데이터 공백일 graceful 처리 필수.

#### P0-2. **VKOSPI + KR 시장폭(breadth)** 도입 → 5팩터의 KR 프록시 불일치 해소
- **문제**: KR 국면이 **미국 VIX**(외국 변동성)에 의존하고 **breadth 팩터가 아예 미투표**. 거시닻 하한(15%)이 외국 VIX 가중을 KR에 강제 → 구조적 불일치.
- **제안**:
  1. **VKOSPI**(코스피 변동성지수) 수집 → KR `vix` 팩터를 VKOSPI 기반으로 교체(미국 VIX는 전염채널로만 유지).
  2. **KR breadth**: 상승종목비율, 20일선 상회 종목%, 신고가-신저가 차이 → `breadth_ratio` KR 산출 → 미투표 팩터 활성화.
- **구현 위치**: `data_updater.py`(VKOSPI/지수폭 수집), `predictive_regime_ensemble.py:884-885`(KR vix/breadth 주입).
- **기대효과**: KR 국면 판정의 **국지적 정확도 급상승**. 현재 4팩터(breadth 빠짐)→5팩터 완전체.
- **난이도**: 중 / **리스크**: 낮음. VKOSPI 소스 확보가 관건(FDR/KRX/대체 ETF 변동성).

#### P0-3. **위험조정 성과지표(Sharpe/Sortino/Calmar) + 팩터 어트리뷰션** 추가
- **문제**: 승률·PF·MDD만 본다. 승률 높아도 변동성·꼬리위험을 못 본다. "기관 초월" 주장하려면 **위험조정 수익**으로 말해야 함.
- **제안**: `forward_report_scalar.py`에 Sharpe/Sortino/Calmar/일별수익변동성/최대연속손실 + **전략별·섹터별·국면별 수익귀속(attribution)** 추가. 리포트에 "리스크 대시보드" 패널 신설.
- **구현 위치**: `reports/forward_report_scalar.py`, `forward/deep_dive.py`(패널).
- **기대효과**: 의사결정 품질↑, 과최적화·운빨 구분, 전략 생존판정의 근거 강화.
- **난이도**: 낮음 / **리스크**: 매우 낮음(리포팅 전용).

### 🟧 P1 (구조적·고임팩트·중위험) — 다음 분기

#### P1-1. **포트폴리오 상관/공분산 오버레이** → 진짜 분산투자 (★최우선 구조 개선)
- **문제**: 분산이 **개수 기반**(20/시장, 2/섹터)뿐. 2차전지 5종이 "다른 섹터"로 분류되면 제약 없이 동시 보유 → 사실상 한 베팅. **기관과 가장 큰 격차.**
- **제안**:
  1. 보유·후보 종목 간 **60일 수익률 상관행렬** 계산.
  2. 진입 게이트에 **상관 캡**: 신규 후보가 기존 보유와 상관 ρ>0.7 이고 같은 클러스터면 사이징 축소 또는 기각.
  3. 포트폴리오 **유효 베팅 수(ENB, effective number of bets)** 모니터링 → 리포트.
- **구현 위치**: 신규 `portfolio_risk_overlay.py`, `forward/shared.py`(진입 게이트), `forward/deep_dive.py`(ENB 패널).
- **기대효과**: 숨은 집중리스크 제거 → 드로다운 급감. **이 한 항목이 MDD를 가장 크게 낮춘다.**
- **난이도**: 중 / **리스크**: 중(사이징에 직접 개입 → 보수적 롤아웃, 섀도우 먼저).

#### P1-2. **포트폴리오 변동성 타게팅** → 그로스 익스포저 동적 조절
- **문제**: ATR은 개별 주수만 정규화. **포트폴리오 목표 변동성**이 없어, 시장 변동성 급등기에도 총 익스포저가 기계적으로 유지.
- **제안**: 목표 연환산 변동성(예: KR 북 15%) 설정 → 실현/예측 포트폴리오 σ로 **전역 켈리 배수 스케일링**(σ_target/σ_realized). 기존 켈리 클러치·둠스데이 댐퍼와 곱셈 결합.
- **구현 위치**: `portfolio_risk_overlay.py`, `meta_governor_consumer.py`(전역 배수 결합).
- **기대효과**: 변동성 급등기 자동 디그로싱 → 꼬리손실 완화, 샤프 개선.
- **난이도**: 중 / **리스크**: 중(기존 다중배수 체인과 상호작용 → 캡/플로어 설계 주의).

#### P1-3. **DART 공시 + 펀더멘털(PER/PBR/실적)** 도입 → 멀티팩터화
- **문제**: 사실상 **순수 기술적**. 가치·퀄리티·실적서프라이즈 신호 0. 기관은 멀티팩터.
- **제안**:
  1. **OpenDART API**(무료): 주요사항보고(공급계약·증자·자사주), **5%·임원 지분공시**(스마트머니 선행), 실적속보.
  2. **펀더멘털**: PER/PBR/ROE/영업이익 증가율 → 가치·퀄리티 팩터. 기술적 신호와 **교차검증 필터**(예: 슈퍼노바 + 흑자전환 = 가산).
  3. 어닝 캘린더 → 실적발표 전후 진입 회피/가속.
- **구현 위치**: 신규 `dart_fetcher.py`, `fundamentals_fetcher.py`, `market_data.sqlite` 신규 테이블, 진입 교차검증.
- **기대효과**: 기술적 함정(부실주 급등) 회피, 알파 다변화, 기관식 멀티팩터 진입.
- **난이도**: 중상 / **리스크**: 낮음(추가 필터, 기존 무영향). API 레이트리밋·키 관리.

#### P1-4. **공매도/대차잔고(공매도 비중·잔고)** 도입 → 숏스퀴즈/크라우디드 숏
- **문제**: 공매도 데이터 0. 숏스퀴즈(급등 연료)·크라우디드 숏(하락 위험) 신호 부재.
- **제안**: KRX 공매도 일별 데이터 적재 → **공매도 급감+가격반등=스퀴즈 후보**, **공매도 급증=경계** 필터.
- **구현 위치**: 신규 `short_interest_fetcher.py`, 진입 가산/경계.
- **기대효과**: 상한가/슈퍼노바 전략과 강한 시너지(스퀴즈는 급등의 핵심 동력).
- **난이도**: 중 / **리스크**: 낮음.

### 🟨 P2 (정교화·검증·장기) — 토대 강화

#### P2-1. **Walk-Forward + Purged/Embargo CV + Deflated Sharpe** 검증 하네스
- **문제**: 국면 분류기·전략 승격에 **walk-forward 없음**. 뮤턴트 OOS는 같은 DB 최근창(시간간격 0)·다중검정 보정 없음 → false discovery.
- **제안**: López de Prado식 **purged k-fold + embargo** 백테스트 하네스, **Deflated Sharpe Ratio**(다중검정 보정), 전략 승격 게이트에 OOS 시간분리 강제.
- **구현 위치**: 신규 `validation/walk_forward.py`, `mutant_oos_validator.py` 강화.
- **기대효과**: "백테스트는 좋은데 라이브는 죽는" 과최적화 근절. 신뢰도 있는 전략만 라이브.
- **난이도**: 상 / **리스크**: 낮음(검증 전용, 라이브 무영향).

#### P2-2. **국면별 kelly_cap 학습화** → 마지막 매직상수 제거
- **문제**: KR 사이징 최대레버인 `ACTION_BY_REGIME.kelly_cap`(BULL 0.028 등)이 **하드코딩**. 게다가 모든 행 `kelly_floor=None` → 위기×클러치×댐퍼 곱셈으로 **0 근처 락아웃** 가능.
- **제안**: 국면별 실현 샤프/손익으로 kelly_cap을 베이지안 밴딧/RL로 자가조정(기존 가중치 진화와 동일 철학), 하한(floor) 도입.
- **구현 위치**: `meta_governor.py`(ACTION_BY_REGIME 동적화).
- **기대효과**: 국면 사이징 최적화, 전환기 과도한 락아웃 방지.
- **난이도**: 상 / **리스크**: 중(핵심 레버 → 섀도우 장기 관측 후).

#### P2-3. **종목별 뉴스 감성 + 통합 NAV** 정교화
- **문제**: 뉴스는 시장 1점/일(종목별 없음). KR/US/테일 자본기준 이원화(₩2천만 vs ₩3억)로 통합 포트폴리오 뷰 부재.
- **제안**: 종목별 헤드라인 감성(이벤트 트리거), **단일 통합 NAV·리스크 예산**(크로스북 상관 반영, 카나리아 발화 시 주식 디그로싱 자동화).
- **난이도**: 상 / **리스크**: 중.

#### P2-4. **체결/유동성 인지 사이징** (가상매매 현실성↑)
- **문제**: 일봉·0.5% 고정 슬리피지. KR 소형주는 ADV 대비 충격 큼.
- **제안**: ADV 참여율 캡(부분적으로 시총 브레이크 존재), 거래대금 대비 동적 슬리피지 모델, (선택) 분봉 진입 타이밍.
- **난이도**: 중상 / **리스크**: 낮음(가상매매 정합성 향상).

## 4. "나만의 비대칭 엣지" — 개인이 기관을 진짜로 이기는 길

기관은 **유동성·집행속도·규모**로 이긴다. 개인은 같은 게임으로는 못 이긴다. 본 시스템이 이미 가진/키워야 할 비대칭 엣지:

1. **소형주 영역 독점** — 기관은 규모 때문에 시총 작은 종목에 못 들어온다. 본 시스템의 KR 소형주 브레이크는 "회피"인데, 오히려 **소형주 전용 알파(상한가 DNA·슈퍼노바)** 를 더 키우는 게 비대칭 우위. (단, P1-1 상관 캡으로 집중리스크는 통제.)
2. **상한가 DNA 포렌식** — 한국 상한가(±30%) 제도는 한국 고유. 폭발 전 DNA 코호트룰은 **기관이 잘 안 하는 영역**. 여기에 P0-1 수급 + P1-4 공매도 스퀴즈를 결합하면 강력.
3. **리테일/수급 타이밍** — 외국인·기관 순매수 시계열(P0-1)은 한국에서 매우 예측력 높은 무료 데이터. 기관은 이미 자기 주문을 알지만, **개인은 "기관 발자국"을 추종**할 수 있다.
4. **24/7 크립토 선행 레이더(카나리아)** — 이미 구축. 크립토 유동성 스트레스를 주식 국면에 선반영하는 건 전통 기관이 잘 안 하는 크로스에셋 엣지.
5. **빠른 진화 사이클** — 기관은 위원회·컴플라이언스로 느리다. 개인 시스템은 매일 RL 진화. **민첩성 자체가 엣지.**

> 전략 요지: **"엣지(소형주·상한가·수급·크립토선행)는 공격적으로 키우고, 리스크(상관·변동성)는 기관 수준으로 방어한다."** 이 비대칭 조합이 기관 초월의 현실적 경로다.

## 5. 발견된 잠재 결함 (수정 아님 — 검토 시 발견, 별도 판단 필요)

탐색 중 발견된, 알파/정합성에 영향 가능한 항목. (사용자 지시에 따라 **수정하지 않음**, 기록만.)

1. **EMA5 DTW 도플갱어 죽은 코드** — `ema5.py`에서 `ns_prefix` 사용-전-할당(use-before-assignment) 버그로 DTW 매칭이 항상 except로 빠짐(무력화). 의도된 동작 아닐 가능성. (`ema5.py:435 vs 477`)
2. **DANTE 원장 미기록** — DANTE는 `try_add_virtual_position` 호출이 없어 텔레그램만 전송, 가상매매 미반영. 의도/누락 확인 필요.
3. **ma200 명명 오해** — `long_trend`의 ma200은 실제 **EMA(200)**. 변수명이 SMA로 오인 소지.
4. **kelly_floor 전무** — `ACTION_BY_REGIME` 모든 행 floor=None → 위기×클러치×둠스데이 곱셈으로 사이징 0 근처 락아웃 가능(P2-2).
5. **자본기준 이원화** — 진입 사이징 ACCOUNT_SIZE ₩2천만 vs NAV 회계 BASE_CAPITAL ₩3억. 의도된 분리인지 확인.
6. **뮤턴트 OOS 시간간격 0** — in-sample 육종 DB와 OOS 검증창이 겹침 + 다중검정 보정 없음 → false discovery(P2-1).
