### 직답
- **"현재 장세가 우리 구조의 상승/하락/횡보와 똑같이 움직이는지"** 를 **측정하는 장치는 아직 없습니다.** 시스템은 장세를 *판정(classify)* 하고, 그 라벨 아래에서 *성과를 집계(condition)* 하지만, **"오늘의 국면 벡터"와 "과거의 특정 국면 에피소드"를 비교해 유사도(닮음) 점수를 산출하는 모듈이 시장국면 레벨에는 부재**합니다.
- 따라서 **"비슷하면 어느 정도, 안 비슷하면 얼마나 안 비슷한지"** 를 답할 수 있는 **정량 근거(상관계수·정확도·거리)가 현재 코드로는 생성되지 않습니다.** 이 측정기가 없으면 "선취매를 해도 되는가"의 통계적 판단 근거도 비어 있습니다.
- **선취매·자가진화 기관급 로직은 실제로 배선되어 작동**하고 있고 **미국/한국 양쪽에 이식**돼 있습니다. 다만 그것들이 **"검증된 장세 유사도"에 게이팅(gating)되어 있지 않고**, 별도의 휴리스틱(섹터 매칭, 컷오프 완화)으로 돌고 있습니다. 즉 "선취매 엔진"은 있는데 "선취매를 켜도 되는지 판정하는 정확도 계측기"가 빠져 있습니다.

## 1. 현재 장세 감지 구조 (상승 · 횡보 · 하락)

장세를 판정하는 엔진은 **총 4개**입니다. 단 1개만 라이브(실전 config에 기록), 나머지 3개는 섀도우(관측 전용).

### 1.1 라이브 분류기 — `regime_meta_analyzer.py` (공식 SSOT)
"지금 시장이 뭐하는가"를 실제로 판정하는 유일한 라이브 엔진. yfinance로 지수를 받아 `REGIME_ANALYSIS` + `CURRENT_REGIME_KEY`를 `system_config`에 기록.

- **입력:** `^GSPC`(미국 프록시), `^KS11`(한국 프록시), `^VIX` — 각 1개월 시계열 (`analyze_market_regime` 320–343행)
- **지표:** 종가, MA20 상회 여부(`above_ma20`), 최근 5영업일 고저폭 %(`range5d_pct`) (`_snapshot_index` 128–153행)
- **임계값** (53–55행):
  - `VIX_HIGH = 20.0`
  - `NARROW_RANGE_PCT = 3.5`
  - `MA_WINDOW = 20`
- **판정 순서** (`_classify_regime` 156–224행, 직접 확인):
  1. 두 지수 모두 조회 실패 → `UNKNOWN`
  2. `VIX > 20` → `HIGH_VOL` (🌪️ 고변동성 발작장)
  3. SPX·KOSPI 둘 다 5일 고저폭 < 3.5% → `SIDEWAYS` (📦 횡보·박스권)
  4. 둘 다 MA20 위 → `BULL` (🔥 상승)
  5. 둘 다 MA20 아래 → `BEAR` (🧊 하락)
  6. 혼재 → `SIDEWAYS` (⚖️ 방향성 탐색)

**한계:**
- **미국·한국이 하나의 글로벌 국면으로 융합**됨 — BULL/BEAR이 되려면 *두 지수가 같은 방향*이어야 하고, 어긋나면 무조건 SIDEWAYS. **라이브 경로에는 KR 단독·US 단독 방향성 국면이 없음.**
- **현재 스냅샷 한 장(`iloc[-1]`)만** 사용 — 시계열을 저장하지 않으므로 **과거와의 비교 불가**.
- **고전 추세 지표 부재:** EMA 크로스, ADX, 시장폭(breadth)이 라이브 분류에 없음 (breadth는 거버너의 진입 티어 컷오프에만 쓰임 `meta_governor.py:933–945`).
- **외부 네트워크 의존:** yfinance 실패 시 `UNKNOWN`으로 강등 → 보수적 동작 + self-heal 루프.

## 2. 【핵심】 "현재 vs 과거 장세 유사도/정확도" — 실제로 있는가?

### 결론: **시장국면 레벨에는 없음. 이것이 가장 큰 공백입니다.**

질문하신 "오늘 시장이 과거 상승/횡보/하락 국면과 *똑같이* 움직이는지, 닮았으면 몇 %인지"를 산출하는 **유사도/거리 점수 모듈이 국면 레벨에는 존재하지 않습니다.** 시스템이 실제로 하는 것은 세 가지로, 모두 "유사도 측정"이 아닙니다:

#### (a) 라벨링 (classify) — "지금은 무슨 국면인가"
→ §1. 현재 한 장의 스냅샷에 BULL/BEAR/SIDEWAYS 딱지를 붙임. 과거와 비교 안 함.

#### (b) 견고성 백테스트 (robustness) — "지금 로직이 과거 국면에서 살아남는가"
**`time_machine_backtester.py`** — 가장 근접하지만 *유사도가 아니라 견고성* 측정.
- `REGIME_PERIODS` (69–88행): 손으로 라벨링한 과거 구간 (2008 서브프라임, 2020 코로나, 2022 금리인상 = `EXTREME_CRASH`; 2020–21 = `MASSIVE_BULL`; 2023 중반 = `CHOPPY_STAGNANT`)
- *현재* 템플릿(`LIVE_CLUSTER_TEMPLATES` 등)을 과거 구간에 재생 → 국면별 승률·PF·평균수익
- ROBUST 판정: 모든 국면에서 `PF > 1.2` (606–621행)
- **명시적 룩어헤드 한계:** 오늘 템플릿을 과거에 적용. config 스냅샷 동결(WFO)은 **스켈레톤/플레이스홀더** — 탐지는 하나 실제로는 현재 config로 fallback (419–452행)
- → "오늘이 과거와 닮았는가"가 아니라 "오늘 로직이 과거였다면 얼마 벌었나"를 측정

#### (c) 국면별 성과 버킷 (condition) — "각 국면에서 어떤 전략이 벌었나"
**`macro_matrix_incremental.py` + `shadow_macro_validator.py`** — `REGIME|arm` 셀에 실현수익 누적(`MIN_CELL_N=3`, 3년 룩백). 국면-조건부 성과 최적화이지 유사도 점수 아님.

### 2.1 "정확도(accuracy)"에 가장 근접한 것 — `meta_learner.py`
유일하게 "어느 국면 예측기가 맞았나"를 채점하지만, *과거와의 닮음*이 아니라 *내부 PRI vs 외부 국면이 어긋난 날, 7일 후 실현 PnL 부호*로 신뢰가중치를 RL 업데이트:
- `W_internal(t+1) = clip(W ± α·tanh(|ΔPnL|/scale), 0.20, 0.80)` (226–234행)
- 발산(divergence)한 날에만 작동, KR+US PnL 합산
- → 국면 콜의 *방향 적중률*은 측정하나 "국면 X와 Y% 닮음"은 아님

### 2.2 "유사도 점수"가 실제로 존재하는 곳 — 단, **종목 DNA 레벨** (국면 아님)
진짜 코사인/유클리드 유사도 기계는 **개별 종목 궤적** 레벨에만 있음:
- `supernova_hunter.py`: 아카이브 DNA 대비 코사인 유사도 `_cosine_sim_archived_dna` (1240–1247행), 라이브 매칭 루프 (1709–1724행)
- `legacy_archive/scanners/master.py` (498–696행), `nulrim.py` (437–560행): 최근 N일 궤적의 7차원 **유클리드 "유사도"** vs 과거 대장주/참사주 DNA → `match_similarity` %, 임계 ≥ 80%, "최근 N일 궤적이 과거 …궤적과 똑같습니다" 메시지까지 존재
- → **이 기계(코사인/유클리드 매칭)는 이미 있으므로, 이를 "국면 벡터" 레벨로 끌어올리면 공백을 메울 수 있음** (핵심 처방, §7)

### 2.3 검증 타깃은 "문서에만" 존재
`Proprietary_Regime_Audit.md`가 스스로 인정:
- "공식 SSOT가 여전히 외부 지수에 기댐"(14행), PRI를 섀도우로 병행→90일 검증 후 교체 권고(23–24행)
- **C-Level 체크리스트(338행): "Shadow 90일: PRI vs 실현 PnL 상관계수 > 0.3 목표"** ← **이 상관계수를 계산하는 코드는 어디에도 없음**
- PRI의 "가상 시뮬 개선"은 고정 배수(`UP:1.08, DOWN:0.92, SIDEWAYS:1.0` `weekly_proprietary_regime.py:347–350`) — **측정된 적중률이 아니라 하드코딩 플레이스홀더**

## 3. 선취매(Front-Running) 이식 현황

선취매는 **3계층**으로 실제 구현되어 있고, 진입 단계(`forward/shared.py`)에서 **컷오프 완화 + 켈리 증액**으로 실제 매매를 바꿉니다.

### 3.1 세력매집 레이더 — `smart_money_tracker.py` (KR+US)
- **KR:** pykrx 외인·기관 순매수 → 다이버전스 필터(5일 +1.8% 이하인데 매집) → **칼만 필터로 세력 평단 추정**(`_kalman_latent_avg_cost_1d` 402행, 연속매집 시 관측노이즈 축소). `SMART_MONEY_RADAR`
- **US:** 다크풀 프록시 — 저변동성 거래량 급증(20일 평균 ≥2배, |변동| ≤2%). `SMART_MONEY_RADAR_US`
- **소비:** `supernova_hunter`가 현재가가 세력평단 3% 이내면 "[세력 매집 포착]" 태그+점수 부스트 (1960–1965행)
- ⚠️ US는 진짜 13F/다크풀 피드가 아니라 **프록시**임을 코드가 명시(605–610행)

### 3.2 섹터 순환매 예측 — `sector_spillover_refresh.py`, `sector_rotation_store.py`
- **Markov 2-step:** 일일 우세섹터 전이행렬 제곱(`M2 = M @ M`)으로 *다음* 주도섹터 예측 → `PREDICTED_NEXT_SECTOR_{KR|US}` (215행)
- 진입 시 `ROTATION_PRE` 태그로 코사인/ML 컷오프 ×0.85 (`scanner_synergy_engine.py:107–112`, **시장 가드 없음 = KR·US 양쪽**)
- US→KR 테마 다리: `us_kr_theme_bridge.py` — US 주도섹터 → KR 하위테마 매수리스트 (`_US_STD_TO_KR_PLAYBOOK`, 단방향)

### 3.3 한도가/폭등 직전 DNA 역설계 — `limit_up_forensics.py` + `forensics_pioneer.py` (KR+US)
- 오늘 한도가/폭등 종목의 T-10~T-2 *폭등 직전 구간* 패턴 추출 → 10개 사전징후(`vol_compression`, `pre_squeeze_narrow_range`, `volume_dry_then_lift` 등)
- 70% 합의(≥7/10) → `pre_emptive_rule` 저장 (`LIMIT_UP_COHORT_DNA_CONFIG_KEY`)
- `forensics_pioneer`가 *오늘* 종목을 이 룰에 매칭 → `virtual_trade_history`에 선취매 기록 (직접적 선취매 액터)

### 3.4 진입 단계 소비 — `forward/shared.py` (선취매가 실제 매매로 바뀌는 곳)
- `is_rotation_prebuy` / `is_spillover_prebuy` (1899–1901행)
- 컷오프 완화: 순환매 ×0.85, 스필오버 ×0.90 (1909–1917행)
- 켈리 부스트: 순환매 ×2.0(`ROTATION_ADVANTAGE_ACTIVE` 시), 스필오버 ×1.5 (2056–2064행)
- 태그 `#순환매_선취매`, `[🌐스필오버 선취매]`
- 주간 검증(`system_auto_pilot.py:1486–1506`): 선취매 트레이드 vs 표준 분리 → 선취매가 이기면 다음 주 켈리 ×2

### 3.5 인접 모듈
- `blackhole_hunter.py` — 역방향 선취매(선제 숏), **US 전용**, 격리된 `short_data.sqlite`

### 4.1 "미래형 퀀트 = 현재 데이터를 닮아간다"는 어디까지 와 있나

- ✅ **포렌식 졸업:** 섀도우 DNA의 Wilson 하한이 라이브 14일 승률을 통계적으로 이기면 라이브 템플릿 승격 (`maybe_graduate_forensics`)
- 🟡 **그러나 "현재 *장세* 자체를 닮아가는 폐루프"는 미완:** 현재 진화는 *종목/전략 DNA* 를 따라감. "현재 국면 벡터 ↔ 과거 국면" 유사도를 측정→그 오차를 줄이는 방향으로 진화하는 루프(§2 공백)가 없으므로, **"안 비슷할 때 현재 장세를 닮아가서 선취매로 이어진다"는 사이클이 장세 레벨에서는 닫히지 않음.**

### 핵심 메시지 (3줄)
1. **"선취매를 켜는 엔진"은 다 있는데, "선취매를 켜도 되는지 판정하는 정확도 계측기(현재 vs 과거 국면 유사도)"가 비어 있습니다.**
